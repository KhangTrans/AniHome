# Moderation Module

## Overview
Kiểm duyệt nội dung và xử lý báo cáo vi phạm.

## Files
- `ContentModerator.jsx` - Review and moderate blog posts
- `ReportManager.jsx` - Handle user reports and violations

## Features

### Content Moderation
- Review pending blog posts
- Approve/reject content
- View post details

### Report Management
- View all user reports
- Handle violations
- Take action on reported content

## API Services
- Uses: `src/services/admin/adminModerationService.js`
