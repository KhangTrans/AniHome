# Shelters Management Module

## Overview
Quản lý các trạm cứu hộ trong hệ thống.

## Files
- `index.jsx` - Main shelter management page

## Features
- Danh sách tất cả shelters với pagination
- Search và filter theo status
- Approve/Reject/Delete shelters
- View shelter details

## API Services
- Uses: `src/services/admin/adminSheltersService.js`

## Components (Future)
- `components/ShelterTable.jsx` - Shelter data table
- `components/ShelterFilters.jsx` - Filter controls
- `components/ShelterDetailModal.jsx` - Detail view modal
