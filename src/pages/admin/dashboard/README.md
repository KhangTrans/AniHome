# Dashboard Module

## Overview
Admin dashboard với thống kê tổng quan hệ thống.

## Files
- `index.jsx` - Main dashboard page với charts và statistics

## Features
- Thống kê người dùng, trạm cứu hộ, thú cưng
- Biểu đồ adoption success rate
- Biểu đồ donation & expense tracker
- Pending blog posts management

## Components (Future)
- `components/StatCard.jsx` - Reusable stat card component
- `components/PieChart.jsx` - Custom pie chart wrapper
- `components/BarChart.jsx` - Custom bar chart wrapper
